无平方因子数

求 nnn 以内的无平方因子数的个数。亦即，求

但是这样多去掉了一些数。比如对于不同的素数 p1,p2，p12p22p_1, p_2，p_1^2p_2^2p1​,p2​，p12​p22​ 的倍数就被去掉了两次，个数是 ⌊np12p22⌋\left\lfloor\frac n {p_1^2p_2^2}\right\rfloor⌊p12​p22​n​⌋ ，应该加回来

显然这是容斥原理

如果 ddd 是 sss 个不同素数的乘积，那么其对答案的贡献是 (−1)s⌊nd2⌋(−1)^s \left\lfloor \frac n {d^2} \right\rfloor(−1)s⌊d2n​⌋ 。 如果 ddd 不是不同素数的乘积，即 ddd 有平方因子，那么 ddd 对答案没有贡献。 容斥的系数恰好是 MobiusMobiusMobius 函数

因此, 答案为:

∑k=1n=∑d=1nμ(d)⌊nd2⌋\sum\limits_{k=1}^n=\sum\limits_{d=1}^{\sqrt n}\mu(d)\left\lfloor \frac n{d^2}\right\rfloor k=1∑n​=d=1∑n​​μ(d)⌊d2n​⌋

事实上，MobiusMobiusMobius 反演本身就可以看成是对整除关系的容斥

即, 我们得到了:

12n(n+1)=∑d=1nϕ(⌊nd⌋)=ϕ(n)−∑d=2nϕ(⌊nd⌋)∴ϕ(n)=12n(n+1)−∑d=2nϕ(⌊nd⌋)\frac12n(n+1)=\sum_{d=1}^n\phi(\left\lfloor\frac nd\right\rfloor)=\phi(n)-\sum_{d=2}^n\phi(\left\lfloor\frac nd\right\rfloor)\\ \therefore \phi(n)=\frac12n(n+1)-\sum_{d=2}^n\phi(\left\lfloor\frac nd\right\rfloor) 21​n(n+1)=d=1∑n​ϕ(⌊dn​⌋)=ϕ(n)−d=2∑n​ϕ(⌊dn​⌋)∴ϕ(n)=21​n(n+1)−d=2∑n​ϕ(⌊dn​⌋)
