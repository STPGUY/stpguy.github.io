初始设计中，微带线材料为铜箔，厚度为 35μm35\mu m35μm (1盎司)，宽度为 1mm1\mathrm{mm}1mm；介质层使用 FR4\text{FR4}FR4 材料，厚度为0.5mm0.5\mathrm{mm}0.5mm，空气腔 长 X 宽 X 高为 16mm×10mm×5mm16\mathrm{mm}\times10\mathrm{mm}\times5\mathrm{mm}16mm×10mm×5mm 。

选择模式驱动求解，空气腔的前后表面设置为波端口激励，求解频率设置为 1GHz1\mathrm{GHz}1GHz 。

  • 定义和使用变量
    • 首先定义两个设计变量 width\text{width}width 和 height\text{height}height ，分别用于表示微带线的宽度和介质层的厚度。
    • 因为微带线始终位于介质层正上方，同时为了让微带线始终位于介质层的中心位置，所以微带线起始点坐标也需要使用变量表示，坐标设置为 (-8mm，-width/2，height\text{-8mm，-width/2，height}-8mm，-width/2，height)。
  • 参数扫描分析
    • 分析微带线的特征阻抗随着微带线宽度 width\text{width}width 和介质层厚度 height \text{height }height 的变化关系。
  • 优化设计
    • 优化设计的目标是:当工作频率为 1GHZ\text{1GHZ}1GHZ 时，在保持介质层厚度 height=0.5mm\text{height=0.5mm}height=0.5mm 不变的情况下，改变微带线宽度 width\text{width}width，使微带线的特征阻抗达到 50Ω50\Omega50Ω。
  • 调谐分析
    • 手动改变变量 height\text{height}height 和 Width\text{Width}Width 的值，使用调协分析功能实时显示求解结果。
  • 灵敏度分析
    • 从优化分析结果可知，在 width=0.806mm\text{width} = 0.806\mathrm{mm}width=0.806mm，height=0.5mm\text{height} = 0.5\mathrm{mm}height=0.5mm 时微带线的特征阻抗约为 50Ω50\Omega50Ω；这里在 width=0.806mm，height=0.5mm\text{width} = 0.806\mathrm{mm}，\text{height} =0.5\mathrm{mm}width=0.806mm，height=0.5mm 附近分析两个变量的变化对微带线的特征阻抗的影响。
  • 统计分析
    • 假设 width\text{width}width 和 height\text{height}height 制造误差为 ±10%\pm10\%±10%，且均匀分布，分析此时微带线的特征阻抗的统计分布情况。
