基本模型

创建基本物体模型

  • 状态栏输入位置坐标及大小尺寸
  • 属性对话框设置位置坐标及大小尺寸

属性对话框

  • Name\text{Name}Name ： 物体的名称。
  • Material\text{Material}Material ： 物体的材料属性。
  • Solve Inside\text{Solve Inside}Solve Inside ：物体内部场的分析。对于介质材料需要分析内部场，而良导体不需要。
  • Orientation\text{Orientation}Orientation ：创建物体的默认参考方向。
  • Model\text{Model}Model ：值为 111 表示为实体物体，值为 000 表示其并非实体物体。
  • Display Wireframe\text{Display Wireframe}Display Wireframe ：物体的显示方式。值为 000 以实体的方式显示，值为 111 以边框的方式显示。
  • Material Appearance\text{Material Appearance}Material Appearance ：采用具有材料特性的外形。
  • Color\text{Color}Color ：物体的颜色。
  • Transparant\text{Transparant}Transparant ：物体的透明度。

可以在 Lines\text{Lines}Lines 下找到一维的线模型，在 Sheets\text{Sheets}Sheets 下找到二维的面模型，在 Solids\text{Solids}Solids 下找到三维模型。

螺旋结构 (Helix\text{Helix}Helix)

创建螺旋结构的操作需要选中一维线模型或者二维平面模型后才能激活，软件以选中的线模型或者面模型为横截面，沿着指定的方向螺旋盘升生成螺旋结构模型，模型底圈的半径是设定的方向矢量到选中的线模型/面模型中心的距离；选中一维线模型生成的是中空的螺旋体，选中二维平面模型生成的是实心的螺旋体。

通过 Sweep\text{Sweep}Sweep 生成模型

物体的材料属性

  • 各项同性材料：
    • 相对介电常数
    • 相对磁导率
    • 电导率
    • 介质损耗正切
    • 磁损耗正切
  • 各项异性材料
  • 铁氧体材料
    • 磁饱和度
    • 朗德因子
    • 磁共振线宽

编辑材料属性

Tools > Edit Libraries > Materials > Add Material...\text{Tools > Edit Libraries > Materials > Add Material...}Tools > Edit Libraries > Materials > Add Material...

将 Type\text{Type}Type 属性修改为 Anisotropic\text{Anisotropic}Anisotropic 后可以修改各向异性材料的属性。

可以看见还有 Tensor\text{Tensor}Tensor 和 Nonelinear\text{Nonelinear}Nonelinear ，分别表示可以以 张量 和 非线性函数 的形式修改这个参数的特性。

但是一般来说用不到这两个选项。
